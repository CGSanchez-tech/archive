# to run

install virtualenv

```bash
pip install virtualenv
```

activate virtualenv

```bash
source ./bin/activate
```

install dependencies

```bash
pip install -r requirements.txt
```

exit virtualenv

```bash
deactivate
```

freeze dependencies

```bash
pip freeze -l > requirements.txt
```
