## Usage

### Install Virtualenv

```bash
pip install virtualenv
```

### Create Virtualenv and Activate

```bash
python3 -m venv .
source ./bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python3 app.py
```

### Freeze Dependencies

```bash
pip freeze -l > requirements.txt
```

### Exit Virtualenv

```bash
deactivate
```
